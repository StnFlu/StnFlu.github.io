Inside Us

Inside Us is a time senstitive 2D survival game where you play as a the lone white blood cell, the player has to defeat the infection whilst the infection simultaneously kills the body. This game was created as part of the Univerty of Lincoln's 2021 Summer Game Jam. C# Recieving 1st Place from community votes. Inside Us was created over a 48 hour time span with the given theme of "Healing", in the Unity game engine using fully original assets. The game can be found here .
